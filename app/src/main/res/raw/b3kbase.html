Diagnostiquer une panne matérielle

La carte mère d’un PC réalise dès la mise sous tension un ensemble d’opérations consistants à interconnecter les composants/éléments matériels : le BIOS (Basic Input Output System) informe l’utilisateur sur l’état des disques durs, lecteurs CD, mémoire vive, processeur, etc. Une pile électrique ronde alimente une mémoire CMOS enregistrant les préférences prédéfinies.

Configuration du BIOS

Il faut un mot de passe pour entrer sur la page d’information du BIOS. Toutefois les pannes matérielles sont détectées avant cette phase du démarrage : un code d’erreur est affiché pour chaque problème rencontré lors de la mise sous tension. Le BIOS n’est accessible que si une mémoire vive est détectée et si les ventilateurs fonctionnent. Si la carte mère est alimentée et que le PC n’affiche pas l’erreur, le processeur est défectueux.

Configuration du boot

Les ordinateurs peuvent démarrer sur le réseau, sur une clé USB, ou sur un disque local. Le premier permet de diagnostiquer une panne du contrôleur Ethernet, car la fonction PXE est native de la carte. Le deuxième test concerne la mise en mémoire des informations de la clé USB, à savoir la fiabilité des calculs du processeur. Enfin le troisième démarrage sur le disque local finit de compléter la liste des problèmes que l’on peut rencontrer au démarrage.

Les affichages vidéos

Il existe plusieurs normes : analogique (VGA), numérique (DVI) et haute définition (DP, HDMI). La connectique est indépendante du contrôleur graphique, qui diffuse aussi bien sur les trois formats. Une carte externe est installée sur la plupart des postes informatiques de projection, tandis que le chipset graphique de la carte mère permet de tester un problème d’affichage.
