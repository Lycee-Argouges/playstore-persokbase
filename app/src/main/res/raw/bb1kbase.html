Installer un logiciel en local

Le manager de logiciels Chocolatey est utilisé pour faciliter la maintenance des logiciels sur les postes de travail. Un package logiciel est appelé sur le site chocolatey.org depuis un terminal en ligne de commande, puis les dépendances sont vérifiées par le système d’installation des packages, qui silencieusement déploie le logiciel dans l’environnement client.

Packages Open Source

Rendez-vous sur le site de l’éditeur pour rechercher les packages disponibles, puis entrez la commande “choco install …" dans une invite de commande administrateur (CMD). Ensuite ce package est maintenu à jour avec la commande “choco upgrade …", et désinstallé avec la commande “choco uninstall …" De nombreuses fonctionnalités ajoutent à Windows la facilité de gestion logicielle disponible avec Linux.

Installation du manager

Ce manager est disponible sur les postes pédagogiques, et s’installe facilement depuis une console PowerShell. Pour cela, activez la stratégie des scripts en tapant “Set-ExecutionPolicy AllSigned” ou “Set-ExecutionPolicy Bypass -Scope Process” pour ne modifier que la stratégie liée à cette ouverture du processus. Puis tapez la commande indiquée sur le site de Chocolatey : le manager est installé dans le dossier C:\ProgramData\chocolatey
