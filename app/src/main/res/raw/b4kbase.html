Diagnostiquer une panne de démarrage

L’ordinateur est sous-tension, le secteur d’amorçage du disque dur est accessible, la page Windows apparaît... Mais un problème est rencontré. Au nouveau démarrage, l’utilitaire de résolution des problèmes est exécuté est le PC affiche une progression qui semble s’éterniser, pour afficher le message suivant : “Aucune solution n’a été trouvée”.

Démarrage de Windows

Le bootstrap contient le fichier Ntdlr qui détient les informations sur le matériel et les pilotes requis lors du chargement de NT. Le premier message d’erreur rencontré peut être : A kernel file is missing from the disk. Il charge également le bootmgr, responsable de l’exécution du BCD (Boot Configuration Data) qui détient le chemin du winload. Sur Windows ces valeurs sont toujours les mêmes... il y a donc peu de risque de trouver un autre message d’erreur à cette étape.

Chargement du kernel

Le fichier Kernel Ntdlr s’assure de la mise en mémoire des files du couple Ntoskrnl.exe et Hal.dll. A ce stade un nouveau message d’erreur peut être rencontré : Windows NT could not start because the following file was missing or corrupt. Le programme cherchera dans le registre Windows les clés HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet indispensables à l’initialisation du noyau. Un écran bleu de la mort peut apparaître à cette étape.

Initialisation du noyau

Une restauration du système est indispensable si le fichier NTDETECT.COM a été corrompu. Celui-ci utilise le BIOS pour connaître la configuration et le matériel présent à travers le CMOS. Les paramètres sont passés au noyau par l’intermédiaire de l’exécutable Ntoskrnl.exe initialisé précédemment. A ce moment la ruche HARDWARE est créée dans le registre Windows pour rendre accessible les pilotes des périphériques.

Chargement des services

Enfin, le processus SMSS.exe est lancé par le système. Il est responsable de la gestion des utilisateurs, de l’ouverture des sessions, et de la mise en disponibilité des ressources apparentées. Il signifie Session Management Subsystem.
